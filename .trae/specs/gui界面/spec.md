# GUI 图形界面功能规范

## 为什么需要此功能

当前程序只支持命令行操作，对非技术人员不友好。增加图形用户界面（GUI）后，用户可以通过可视化界面修改配置参数、浏览日志，直接点击按钮执行盖章操作。

## 变更内容

- 使用 Tkinter 构建桌面 GUI 应用
- 界面分为上下两部分：配置面板 + 日志显示
- 参数修改自动保存到配置文件
- 实时显示运行日志

## 影响范围

- 新增 GUI 相关代码
- 配置文件（config.yaml）结构不变
- 原有命令行功能保持不变

## UI 设计

### 界面布局

```
+--------------------------------------------------+
|  AI PDF Seal - 批量盖章工具                       |
+--------------------------------------------------+
|  [目录选择]  _________________________ [浏览]     |
|                                                  |
|  印章图片:  _________________________ [浏览]     |
|                                                  |
|  宽度:    [____50____]  高度:   [____50____]   |
|                                                  |
|  X坐标:   [____450____]  Y坐标:   [____150____] |
|                                                  |
|  [x] 强制覆盖已盖章文件                           |
|                                                  |
|            [    立即执行    ]                    |
+--------------------------------------------------+
|  运行日志:                                        |
|  +--------------------------------------------+ |
|  | [INFO] AI PDF Seal 开始运行                | |
|  | [INFO] 配置文件: config.yaml               | |
|  | [INFO] 开始批量处理，共 3 个文件           | |
|  | [INFO] [1/3] 处理: doc1.pdf               | |
|  | ...                                        | |
|  +--------------------------------------------+ |
+--------------------------------------------------+
```

### 组件说明

| 组件 | 类型 | 说明 |
|------|------|------|
| 目录选择 | 输入框 + 按钮 | 选择 PDF 文件所在目录 |
| 印章图片 | 输入框 + 按钮 | 选择印章图片文件 |
| 宽度/高度 | 数字输入框 | 印章尺寸（像素） |
| X/Y坐标 | 数字输入框 | 印章位置 |
| 强制覆盖 | 复选框 | 是否覆盖已盖章文件 |
| 立即执行 | 按钮 | 开始执行盖章操作 |
| 运行日志 | 文本框（只读） | 实时显示运行日志 |

## 功能需求

### 1. 配置参数管理

- 启动时自动加载 config.yaml 中的配置
- 参数修改后自动保存到配置文件
- 支持目录选择和文件选择的对话框

### 2. 执行盖章

- 点击"立即执行"按钮开始处理
- 处理过程中实时显示日志
- 处理完成后显示汇总信息
- 执行过程中按钮禁用，防止重复点击

### 3. 日志显示

- 使用滚动文本框显示日志
- 自动滚动到最新日志
- 日志内容可复制

## 技术方案

### 框架选择

- **Tkinter**: Python 内置 GUI 库，无需额外安装

### 代码结构

```
main_gui.py          # GUI 入口
src/
  pdf_processor.py   # 原有盖章逻辑（不变）
```

### 日志集成

- 重定向日志输出到 GUI 文本框
- 使用自定义 Handler 实现
