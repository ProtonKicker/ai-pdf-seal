# PDF 图片盖章功能规格说明

## 原因
用户需要对 PDF 文件进行盖章处理，章子使用图片文件呈现。需要支持控制章子大小和在 PDF 页面中的位置，并且能够对 PDF 的每一页都加上相同位置和大小的章。

## 变更内容
- 新增 PDF 图片盖章功能
- 支持通过参数控制章子图片的大小
- 支持通过参数指定章子在 PDF 页面中的位置（X, Y 坐标）
- 支持对 PDF 的每一页都添加相同位置和大小的章

## 新增需求

### 需求：PDF 图片盖章功能
系统应提供对 PDF 文件添加图片章的功能。

#### 场景：成功盖章
- **当** 用户提供 PDF 文件路径、图片章文件路径、章大小参数、位置参数
- **则** 系统在 PDF 的每一页指定位置添加指定大小的图片章，并输出盖章后的 PDF 文件

#### 输入参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pdfPath | string | 是 | PDF 文件路径 |
| imagePath | string | 是 | 章图片文件路径 |
| width | number | 是 | 章宽度（像素或 PDF 单位） |
| height | number | 是 | 章高度（像素或 PDF 单位） |
| x | number | 是 | 章左上角 X 坐标 |
| y | number | 是 | 章左上角 Y 坐标 |
| outputPath | string | 否 | 输出文件路径（默认在原文件名前加 "_sealed"） |

#### 坐标系统
- 原点为 PDF 页面左下角
- X 轴向右为正方向
- Y 轴向上为正方向
